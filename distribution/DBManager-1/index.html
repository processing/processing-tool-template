DB Manager

A tool by Venerando Solis Barrado for the Processing programming environment.
Last update, 04/04/2017.

Tool to manage a Java DB (aka Derby) server in mode "embedded server".
Java stuff over Java DB. It must work in any platform (not tested).

Download

Download DB Manager version 1.0.0 (1) in .zip format.

Installation

Unzip and put the extracted DBManager folder into the tools folder of your Processing sketches. Reference and examples are included in the DBManager folder.

Keywords. Utility,DB,Data,GUI
